# SOP-INC — 개재물(inclusion) 결함 처리 지침

## 정의
강재 표면/내부에 비금속 개재물(산화물·슬래그 등)이 박혀 점·선 형태로 드러난 결함.

## 원인
- 제강 단계의 탈산 불충분, 슬래그 혼입
- 연주 시 보호 분말 혼입

## 판정 기준
- 개재물 군집이 직경 2mm 이상이면 불량.
- 단일 미세 개재물은 경미로 분류하되 빈도가 높으면 상위 심각도로 격상.

## 조치(SOP)
1. 제강/연주 공정 파라미터(탈산제 투입량, 턴디시 온도)를 점검한다.
2. 동일 히트(heat) 번호에서 반복되면 해당 로트를 격리한다.
3. 재발 시 품질팀에 근본원인분석(RCA)을 요청한다.
