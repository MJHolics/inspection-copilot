# SOP-SCR — 스크래치(scratches) 결함 처리 지침

## 정의
표면에 선형으로 긁힌 흔적. 이송 롤러·핸들링 과정의 기계적 접촉으로 발생한다.

## 원인
- 이송 롤러 표면의 이물·돌기
- 코일 권취/권출 시 층간 슬립
- 작업자 핸들링 중 공구 접촉

## 판정 기준
- 길이 5mm 이상 또는 깊이가 도금층을 관통하면 불량(reject).
- 길이 5mm 미만의 미세 스크래치는 경미(minor)로 분류해 재검토.

## 조치(SOP)
1. 해당 라인 정지 후 이송 롤러 표면을 점검·청소한다.
2. 동일 결함이 3건 이상 연속이면 롤러 교체를 요청한다.
3. high 심각도는 사람 검토(reviewed)를 거쳐 출하 보류한다.
