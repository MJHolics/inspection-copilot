# SOP-GEN — 표면 검사 일반 기준 및 심각도 등급

## 심각도(severity) 등급
- **low**: 기능·외관에 영향 미미. 기록 후 통과 가능.
- **medium**: 등급 강등 또는 재작업 검토 대상.
- **high**: 출하 보류. 반드시 사람 검토(reviewed=1)를 거친다.

## 신뢰도(confidence)와 사람 검토
- 모델 신뢰도가 임계(예: 0.70) 미만이거나 분포 밖(OOD)으로 판정되면
  자동 판정을 신뢰하지 말고 사람 검토로 라우팅한다("모를 때 멈춤").
- high 심각도는 신뢰도와 무관하게 항상 사람 검토를 거친다.

## 공통 처리 흐름
1. 결함 클래스·심각도·신뢰도를 기록한다.
2. 판정 기준에 따라 통과/재작업/불량으로 분류한다.
3. 동일 결함이 연속·다발이면 해당 라인·로트를 점검·격리한다.
4. 반복 결함은 근본원인분석(RCA)으로 이관한다.
